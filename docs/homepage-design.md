# 首页设计文档

## 概述

首页采用**手绘艺术风格**设计，融合代码与艺术的理念，呈现一个独特的个人开发者主页。整体设计强调：

- 手绘感 (Sketch Style)
- 艺术与代码的交融
- 简约而不简单
- 一屏高度展示

## 设计元素

### 1. 字体

| 用途 | 字体 | 类名 |
|------|------|------|
| 英文手写 | Caveat | `font-sketch` |
| 中文手写 | Ma Shan Zheng / LXGW WenKai | `font-sketch-cn` |

### 2. 颜色方案

- **亮色模式**: `bg-light-100`, `text-dark-800`
- **暗色模式**: `bg-dark-900`, `text-light-200`
- 使用 UnoCSS `dark:` 前缀实现自动切换

### 3. 手绘 SVG 元素

- **头像边框**: 不规则手绘圆形
- **手绘猫咪**: 坐着的猫咪插画，带胡须和尾巴
- **波浪分隔线**: 手绘曲线装饰
- **导航图标**: 波浪、方框、圆圈、地图标记等

## 技术实现

### 数据来源

- GitHub API (`/users/iceywu`) - 用户信息
- GitHub API (`/users/iceywu/repos`) - 仓库列表、语言统计、Star 数

### 样式框架

- **UnoCSS** - 原子化 CSS
- **自定义 shortcuts**:
  - `font-sketch` - 英文手写字体
  - `font-sketch-cn` - 中文手写字体

### 组件依赖

- `GithubIcon.vue` - 手绘 GitHub 文字 SVG
- `DrakToggle.vue` - 暗色模式切换按钮

### 响应式设计

- 桌面端: 两栏布局 (`grid-cols-2`)
- 移动端: 单栏布局 (`grid-cols-1`)

## 设计理念

> **"代码与艺术的交汇处"**

这个首页的设计核心是打破传统开发者主页的刻板印象，用手绘风格传达：

1. **温度** - 手写字体和手绘插画带来人情味
2. **独特性** - 不规则的边框和歪斜的元素打破对称
3. **诗意** - "Code is poetry" 不只是口号，更是设计语言
4. **留白** - 恰到好处的呼吸空间
